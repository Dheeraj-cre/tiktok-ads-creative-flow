# TikTok Ads Creative Flow (Frontend Assignment)

## Overview
This project is a simplified frontend application that simulates a TikTok Ads
creative setup flow. The goal of this assignment is not to build a full Ads
Manager, but to demonstrate OAuth integration, form validation, conditional
logic, and clear user-friendly error handling.

---

## Tech Stack
- React (Vite)
- React Router
- JavaScript
- LocalStorage (for OAuth token simulation)
